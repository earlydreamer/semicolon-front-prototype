# 18. Shop 페이지 상태별 탭 및 UI 개선

## 개요
Shop 페이지에 판매자 페이지와 동일한 상태별 탭 구조를 적용하고 UI 시인성을 개선했습니다.

## 변경 사항

### 수정된 파일
- `src/pages/ShopPage.tsx` - 탭 구조 및 레이아웃 개선
- `src/components/features/shop/ShopProductList.tsx` - 상태 태그 표시
- `src/mocks/products.ts` - s1 상품 및 s21, s22 seller 추가
- `src/mocks/users.ts` - s21, s22 상점 정보 추가

### 주요 개선 내용

1. **탭 구조 개선**
   - 전체/판매중/예약중/판매완료 탭 (판매자 페이지와 동일)
   - 탭 배경색 추가로 시인성 향상

2. **상품 상태 표시**
   - 모든 상품에 상태 태그 표시 (판매중, 예약중, 판매완료)

3. **레이아웃 개선**
   - 거래 후기를 별도 섹션으로 분리

4. **Mock Data 추가**
   - s1 상점: 상품 7개 (판매완료 3, 예약중 2, 판매중 2)
   - s21, s22: 빈 상점 (판매 상품 없음)
