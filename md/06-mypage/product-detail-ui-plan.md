# [기능] 상품 상세 페이지 상태 태그 세분화 및 배송비 표시 작업 계획

## 개요
상품 상세 페이지에서 상품의 상태와 배송비 정보를 더 명확하게 제공하여 사용자의 구매 결정을 돕습니다.

## 작업 내용
1. **상품 상태 태그 세분화**
   - `SEALED`: 미개봉 (에메랄드)
   - `NO_WEAR`: 사용감 없음 (블루)
   - `MINOR_WEAR`: 사용감 조금 (옐로우)
   - `VISIBLE_WEAR`: 사용감 많음 (오렌지)
   - `DAMAGED`: 하자 있음 (레드)
2. **배송비 정보 노출**
   - 가격 하단 또는 우측에 배송비 상태 표시
   - 무료 배송 시 강조 텍스트 적용

## 검증 계획
- 상태별 태그 컬러 및 텍스트 매핑 확인
- 배송비 유/무에 따른 표시 대응 확인
