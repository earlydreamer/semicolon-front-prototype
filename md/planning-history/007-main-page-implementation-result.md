# 메인 페이지 및 상품 탐색 구현 내역 (Main Page Implementation History)

**날짜**: 2026-01-08  
**작성자**: Semi (AI Agent)

## 개요
번개장터 레퍼런스를 참조하여 메인 페이지의 레이아웃을 구현하고, 공통 디자인 시스템을 적용했습니다. 백엔드 API 부재로 인해 Mock 데이터를 사용하여 UI 및 인터랙션을 검증했습니다.

## 주요 구현 사항

### 1. 메인 배너 (`HeroBanner.tsx`)
- **디자인**: 그라디언트 배경(`from-primary-50 to-primary-100`)과 타이포그래피를 활용하여 브랜드 아이덴티티 강조.
- **구성**:
    - 헤드라인 및 서브텍스트
    - CTA 버튼 (거래 시작하기, 내 물건 팔기)
    - 우측 대표 이미지 카드 (호버 인터랙션 포함)

### 2. 가로 상품 카드 (`ProductCard.tsx`)
- **정보**: 썸네일 이미지, 상품명(2줄 말줄임), 가격(Locale String 적용), 상대 시간(N시간 전), 지역 정보.
- **기능**:
    - `Link` 컴포넌트로 감싸 상세 페이지 이동 지원 (현재는 경로만 설정됨).
    - 안전결제 가능 상품일 경우 이미지 좌측 상단에 배지 표시.
    - 호버 시 이미지 줌인(`scale-110`) 및 그림자 효과로 클릭 유도.

### 3. 상품 목록 그리드 (`ProductList.tsx`)
- **반응형 레이아웃**: Tailwind CSS Grid 활용.
    - 모바일: 2열 (`grid-cols-2`)
    - 태블릿: 3~4열
    - 데스크탑: 5~6열 (`grid-cols-5`, `grid-cols-6`)
- **Mock 데이터**: `src/mocks/products.ts`에 10개의 가상 상품 데이터 정의.

### 4. 홈 페이지 (`HomePage.tsx`)
- `HeroBanner`와 `ProductList`를 조합하여 메인 페이지 구성.
- 라우팅: `App.tsx`에서 `/` 경로를 `HomePage`로 설정하고, 기존 디자인 시스템 페이지는 `/design`으로 이동.
